Information du projet

  • Catégorie : Développement front et back
  • Date : Novembre 2022
  • Technologies : Symfony, Twig, Docker

Description

Création d'une application LeBonCoin simplifiée avec gestion d'annonces, de tags, de profils utilisateurs, de questions/réponses et d'un système de notation pour les vendeurs. Réalisation avec Symfony et Twig, la base de données sera générée via des Fixtures et sera stockée dans un container Docker. Ce projet m'a permis de découvrir le framework Symfony.
